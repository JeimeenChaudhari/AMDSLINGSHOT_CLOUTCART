# Icon Creation Guide

## Required Icons

You need to create 3 icon sizes for the extension:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Quick Icon Creation Options

2. Search for "shopping bags" or "robot"
3. Download the generated icons
4. Rename them to icon16.png, icon48.png, icon128.png

## Temporary Solution

If you need to test immediately, you can:
1. Create any 128x128 PNG image
2. Copy it 3 times
3. Resize to 16x16, 48x48, and 128x128
4. Name them accordingly

The extension will work without icons, but Chrome will show a default placeholder.
