## Installation Steps (5 minutes)

### Step 1: Download the Extension
- Download or clone this repository to your computer
- Extract the ZIP file if downloaded

### Step 2: Open Browser Extensions Page

**For Chrome:**
1. Open Chrome browser
2. Type `chrome://extensions/` in the address bar
3. Press Enter

**For Edge:**
1. Open Edge browser
2. Type `edge://extensions/` in the address bar
3. Press Enter

### Step 3: Enable Developer Mode
- Look for "Developer mode" toggle in the top-right corner
- Click to enable it

### Step 4: Load the Extension
1. Click "Load unpacked" button
2. Navigate to the folder containing this extension
3. Select the folder (the one with `manifest.json`)
4. Click "Select Folder"

### Step 5: Verify Installation
- You should see the extension card appear
- Look for "Emotion-Adaptive Shopping Assistant"
- The extension icon should appear in your browser toolbar

## First Time Setup

### Configure Your Preferences

1. **Click the extension icon** in your toolbar
2. **Choose emotion detection mode**:
   - Enable webcam for real-time emotion detection
   - OR enable keyboard mode for privacy
3. **Toggle features** as needed:
   - Focus Mode (blur ads)
   - Price History
   - Price Comparison
   - AI Recommendations
   - Review Checker

### Test the Extension

1. **Visit a shopping website**:
   - Go to Amazon.com, Walmart.com, or eBay.com
   - Search for any product

2. **Look for the floating panel**:
   - Bottom-right corner of the page
   - Shows your current emotion
   - Displays shopping insights

3. **Check product page features**:
   - Price history widget
   - Price comparison table
   - AI buy/not buy recommendation
   - Review authenticity checker

## Troubleshooting

### Extension not appearing?
- Make sure Developer Mode is enabled
- Try refreshing the extensions page
- Reload the extension

### Features not working?
- Refresh the shopping website page
- Check if the website is supported
- Open browser console (F12) for error messages

### Webcam not working?
- Allow camera permissions when prompted
- Check browser camera settings
- Use keyboard mode as alternative

### Floating panel not showing?
- Scroll down the page
- Check if you're on a product page
- Try refreshing the page
